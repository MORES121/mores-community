# 🏗️ MORES 叉车冠军方案

## 树根杯 · 具身智能平衡重叉车挑战赛

### 技术特色
- 🧠 **因果推理**：基于因果图建模，决策可解释
- 🔮 **反事实推演**：模拟多种行动方案，选择最优
- 👁️ **多模态感知**：RGB + 激光雷达融合
- 🎮 **阿克曼控制**：精确底盘控制 + PID

### 快速启动
```bash
# 安装依赖
pip install -r requirements.txt

# 运行主程序
python code/main.py --config configs/default.yaml
```

### 目录结构
```
├── code/          # 核心源代码
├── configs/       # 配置文件
├── models/        # 预训练权重
├── logs/          # 决策日志
├── materials/     # PPT/视频素材
└── docs/          # 技术文档
```
